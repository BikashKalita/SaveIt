## 📒 A Simple offline Note keeper web-app built using ReactJS

# Features

✔️ **Worked Offline**: All notes are saved in browser localStorage.

✔️ **Ease to Use**: Simple and easy to use.

✔️ **Ligitweight**: Minimal and lightweight design.
